Description du projet

Projet de détection de cyberattaques utilisant l'intelligence artificielle pour analyser des millions de logs système. Suite à une attaque de type ransomware (Sodinokibi), nous avons développé une solution basée sur des algorithmes de machine learning pour identifier les points d'entrée des hackers et retracer l'attaque.

Note : Le code source de ce projet n'est pas disponible publiquement car il est soumis à des clauses de confidentialité.

Technologies utilisées

  • Python
  • Algorithmes d'IA (K-means, Random Forest)
  • Analyse de données massives
  • Cybersécurité

Résultats

  • Identification réussie des points d'entrée des hackers
  • Présentation des résultats au PDG du groupe industriel
  • Amélioration significative de la sécurité du système
  • Publication d'un article technique
  • Présentation de l'approche dans une webconférence en Chine
